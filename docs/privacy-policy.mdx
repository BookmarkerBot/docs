Bookmarker does not collect or store any information. When a user bookmarks a message, certain data is taken and sent to your DMs such as: Message Author ID, Message Author profile picture, server and channel information, message contents. None of this information is stored with bookmarker. All of the information remains in the direct messages of the user and bot on Discord.

Bookmarker is open source, you can find the repository here: https://github.com/Squidtoon99/bookmark-bot

Contact Information: If you have any questions or concerns regarding the privacy policy, please contact us at dakshdiscordgg@gmail.com or via the contact form: https://tally.so/r/w7q1EA/
